# Expert Evaluation Rubric for WRITER-Generated Research Reports

## Instructions for Evaluators

You will evaluate AI-generated research reports produced by the WRITER framework. For each report, you were given the original research question that prompted the report. Please rate each dimension on a 1-5 scale using the descriptions below.

---

## Evaluation Dimensions

### 1. Correctness (Factual Accuracy)
*Are the claims and statements in the report factually accurate based on your domain knowledge?*

| Score | Description |
|-------|-------------|
| 5 | All claims appear factually accurate; no errors detected |
| 4 | Minor inaccuracies that do not affect the overall conclusions |
| 3 | Some factual errors present, but the main arguments hold |
| 2 | Several factual errors that undermine parts of the analysis |
| 1 | Pervasive factual errors; the report is unreliable |

### 2. Coverage (Completeness)
*Does the report address the research question comprehensively? Are the major aspects and perspectives covered?*

| Score | Description |
|-------|-------------|
| 5 | Thorough coverage; all major aspects and perspectives addressed |
| 4 | Good coverage; most important aspects included, minor gaps |
| 3 | Adequate coverage; addresses the core question but misses some important areas |
| 2 | Partial coverage; significant aspects of the question are missing |
| 1 | Very incomplete; fails to address the question meaningfully |

### 3. Coherence (Organization and Flow)
*Is the report well-organized? Does the argument flow logically from section to section?*

| Score | Description |
|-------|-------------|
| 5 | Excellent organization; clear logical flow; well-structured sections |
| 4 | Good organization; mostly logical flow with minor structural issues |
| 3 | Adequate organization; some sections feel disjointed or repetitive |
| 2 | Poor organization; difficult to follow the argument; significant repetition |
| 1 | Incoherent; no discernible logical structure |

### 4. Usefulness (Practical Value)
*Would a researcher find this report useful as a starting point for understanding the topic or identifying relevant literature?*

| Score | Description |
|-------|-------------|
| 5 | Highly useful; could serve as a solid foundation for further research |
| 4 | Useful; provides valuable insights and good entry points into the literature |
| 3 | Moderately useful; provides some value but requires significant supplementation |
| 2 | Limited usefulness; a researcher would be better off starting from scratch |
| 1 | Not useful; misleading or too superficial to provide any research value |

### 5. Citation Quality (Source Support)
*Are claims properly supported by cited sources? Do citations appear relevant and appropriate?*

| Score | Description |
|-------|-------------|
| 5 | Claims are well-supported; citations appear relevant and appropriately placed |
| 4 | Most claims are supported; citations are generally appropriate |
| 3 | Some claims lack citation support; some citations may be tangentially relevant |
| 2 | Many unsupported claims; citations are frequently missing or seem irrelevant |
| 1 | Very poor citation practices; most claims are unsupported |
